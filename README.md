# Tuniverse 🌍

**Tuniverse** is a web platform that connects local buyers in Tunisia with travelers abroad who can bring requested products.  
It offers a faster, cheaper, and more reliable alternative to international shipping, leveraging the Tunisian diaspora and frequent travelers.

## 🚀 Features
- Post product requests as a buyer
- Travelers list their routes and available luggage space
- Matching system connects buyers and travelers
- Secure in-app communication
- Payment held in escrow until delivery confirmation
- Ratings and reviews for trust-building

## 🧩 Technologies Used
- **Frontend:** React, React Router
- **API Requests: Axios
- **Backend (planned):Node.js
- **Database (planned):MySQL
