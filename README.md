# Tuniverse 🌍

**Tuniverse** is a web platform that connects local buyers in Tunisia with travelers abroad who can bring requested products.  


## 🚀 Features
- Request products as a buyer
- Accept / reject requests as a seller
- Secure in-app communication

## 🧩 Technologies Used
- **Frontend:** React, React Router
- **API Requests: Axios
- **Backend (planned):Node.js
- **Database (planned):MongoDb

## ⚙️ Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/yasqdb/Tuniverse.git
2. Navigate into the project directory:
   ```bash
   cd Tuniverse
3. Install dependencies:
   ```bash
   npm install
4. Start the development server:
   ```bash
   npm start
